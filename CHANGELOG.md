## [0.1.0] - 2025-11-15

- Initial release of the `compose-to-quadlet` Ansible role.
- Custom Ansible module `compose_to_quadlet` to wrap the Python library.
- Tasks for cloning the Python library and generating Quadlet units.
- Documentation (README, CHANGELOG, LICENSE).
- Default variables for role configuration.
